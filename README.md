# 南华寺项目

这是一个使用 Vue3 + Vite 开发的南华寺景区展示网站。该项目旨在为游客提供一个直观、现代化的南华寺景区在线导览平台。

## 项目特点

- 📱 完全响应式设计，支持移动端、平板和桌面端
- 🎥 高质量视频展示功能
- 🖼️ 景点图片网格展示
- ⚡ 基于 Vite 的快速开发体验
- 🎨 使用 Tailwind CSS 实现现代化 UI 设计

## 技术栈

- Vue 3
- Vite
- Tailwind CSS
- Vue Router
- Vue 3 Composition API

## 开始使用

### 环境要求

- Node.js 18.0 或更高版本
- npm 7.0 或更高版本

### 安装步骤

1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```

## 项目结构

```
nanhua-temple/
├── public/          # 静态资源
│   ├── assets/      # 项目资源文件

│   ├── views/       # 页面视图
│   ├── router/      # 路由配置
│   └── App.vue      # 根组件
```
